# TaxLK — Sri Lankan Income Tax Calculator 🇱🇰

TaxLK is a modern, high-performance web application designed for calculating and managing Sri Lankan personal income tax. Built for accuracy, user experience, and visual excellence.

🚀 **Live at:** [https://taxlk-13159.web.app](https://taxlk-13159.web.app)

## ✨ Features

- **Real-time Tax Calculation**: Based on the 2024 Sri Lankan progressive tax slabs.
- **APIT Breakdown**: Automatically calculates monthly Advanced Personal Income Tax (APIT) proportional to employment income.
- **Multi-Income Support**: Track income from Employment, Business, and Investments.
- **Tax History**: Save snapshots of calculations to track your financial metrics over time.
- **Modern UI/UX**: Premium institution-grade design with high-quality 3D visuals and institutional trust.
- **Firebase Integrated**: Securely synchronized with Firebase Auth and Firestore for persistence.

## 🛠️ Tech Stack

- **Framework**: Vite + React + TypeScript
- **Styling**: Tailwind CSS + Shadcn/UI
- **Animations**: Framer Motion
- **Backend**: Firebase (Auth, Firestore, Hosting, Analytics)
- **Deployment**: Firebase Hosting

## 🏗️ Getting Started

### Prerequisites
- Node.js (v18+)
- npm or bun

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Deployment
To build and deploy to Firebase:
```bash
npm run build
firebase deploy
```
